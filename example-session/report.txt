```
SMTP Input (port 25)
    ↓
smtpd process (SMTP daemon, TLS handling)
    ↓
cleanup service (format validation, header processing)
    ↓
qmgr service (queue manager, retry logic)
    ↓
Delivery agents (depends on recipient):
  - local: /var/mail/%u file delivery
  - virtual: Domain-based virtual mapping
  - smtp: Remote SMTP delivery
  - maildrop: External delivery program
  - lmtp: LMTP protocol to other servers
    ↓
bounce/trace/error services (NDR handling)
    ↓
Remote recipient or local file storage
```

**Queue Structure:**
- `incoming/`: New messages awaiting cleanup
- `active/`: Currently being delivered
- `deferred/`: Delivery deferred, retry scheduled
- `bounce/`: Undeliverable messages being returned
